Detailed Description

template<typename OneCmperBase>
class musx::dom::OneCmperPool< OneCmperBase >

A pool that manages collections of OneCmperBase objects, organized by XML node names and Cmper values.

Examples of OneCmperBase classes are OthersBase and TextsBase.
